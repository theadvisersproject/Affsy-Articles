And there are many more. But if you want to install the JDK on your computer, follow the steps below. JDK is available to download for free at www.oracle.com -> Java SDKs and Tools -> Java SE. Or you can click here to get redirected to downloading page.
Step 1.  Download JDK through the website stated above. Step 2. Click the Accept License Agreement checkbox. And choose your platform from the list.

JDK & PATH

Step 1. On windows, the JDK will install by default in the directory "C:\Program Files\Java\jdk1.8.0_xx" with the latest upgrade number indicated by xx.
  • JRE defaults to "C:\Program Files\Java\jre1.8.0_xx".
Step 2. Open the JDK folder, open the bin folder, and copy the path, which you will need to use in the next step. Three. For the JDK to work, you need to add its parth to the PATH environment variable. Step 4. To do that, right click on my computer and go to properties > Advanced system settings. Step 5. Switch to the Advanced tab > Environment Variables and click on New button if PATH does not exist in the list. If PATH already exist, select it and click edit. Six. Add the copied path value to the variable after ; (semicolon) . Step 7. You should just see "New Variable User" window while adding the PATH. Step 8. After adding, save changes. Final Step. To Verify that JDK is properly install, open the command prompt, type in javac, and press enter. If the JDK has been successfully install, you will see some lines of text inside the command prompt.
And you're ready to move forward !
